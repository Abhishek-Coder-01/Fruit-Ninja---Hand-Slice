# 🍉 Fruit Ninja - Hand Slice

A fun browser-based **Fruit Ninja style game** where you slice fruits using your **hand gestures**.  
This project uses **HTML, CSS, and JavaScript** with **MediaPipe Hands** for real-time hand tracking.

## 🚀 Live Demo
🔗 [Play Now](https://abhishek-coder-01.github.io/Fruit-Ninja---Hand-Slice/)

---

## 📌 Project Overview

**Fruit Ninja - Hand Slice** is an interactive web game in which the player uses their **index finger** in front of the webcam to slice flying fruits on the screen.  
The game includes:

- 🍎 Fruit slicing with hand tracking
- 💣 Bomb avoidance
- ❤️ Lives system
- 🔥 Combo scoring
- ⚡ Special power-ups
- 🎵 Sound effects
- 📱 Mobile landscape support

This project is built only with frontend technologies and runs directly in the browser.

## 🎮 How to Play

1. Open the game in your browser.
2. Allow **camera permission** when prompted.
3. Show your **index finger** in front of the camera.
4. Move your finger to slice fruits.
5. Avoid slicing **bombs**, otherwise you lose a life.
6. Make continuous slices to build **combos** and get a higher score.
